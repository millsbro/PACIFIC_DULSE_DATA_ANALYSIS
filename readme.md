# Pacific Dulse Seasonal Biochemistry Analysis

Analysis pipeline and manuscript figures for seasonal biochemical profiling of cultivated *Devaleraea mollis*.

## Contents

- Abiotic analysis
- Macronutrient composition
- Mineral PCA
- Amino acid PCA
- Bioactive metabolite analysis
- C:N vs FAA analysis

## INSTRUCTIONS

-In R studio, set this directory as wd. Import .R script and all .csv files
-Run pipelin_script_v2.2.R user must select Seasonal or Montlhy mode
-A new folder "figures" is created in wd, all oputputs go here
- All figures in the paper "Seasonal Variation in the Biochemical Profile of Devalaraea mollis (Pacific Dulse)" originated here
  other figures not shown in the paoer are prepared as well.
-Statistics from pearson tests are printed to screen
